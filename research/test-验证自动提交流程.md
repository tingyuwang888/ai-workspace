# AI Workspace

AI Agent 工作产出存档仓库。

## 目录结构

```
ai-workspace/
├── projects/          # 项目相关产出（代码、配置、脚本等）
├── reports/           # 报告、文档、分析结果
├── research/          # 调研、学习笔记
├── scripts/           # 工具脚本、自动化脚本
└── misc/              # 其他杂项
```

## 提交规范

- 每次 AI 任务完成后自动提交
- 提交信息格式：`[日期] 任务简述`
- 例如：`[2026-07-07] 生成三峡部署方案`
