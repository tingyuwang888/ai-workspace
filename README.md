# AI Workspace

AI Agent 工作产出存档仓库。

## 目录结构

```
ai-workspace/
├── <项目名>/
│   └── YYYY-MM-DD/
│       └── 文件...
├── 三峡项目/
│   └── 2026-07-07/
└── 日常/
    └── 2026-07-07/
```

规则：第一层按项目名分类，第二层按日期归档，文件放在日期目录下。
不归属于特定项目的文件统一放入 `日常/` 目录。

## 提交规范

- 每次 AI 任务完成后自动提交
- 提交信息格式：`[YYYY-MM-DD] 项目名 - 任务简述`
- 例如：`[2026-07-07] 三峡项目 - 生成部署方案`
